Forecast UI — three options over the same JSON

All three read the identical forecast.json and recompute nothing — the scores are precomputed by the agent. Each has a dataset switcher: the Hurricane Lee replay has actual surf in it, the live forecast is whatever the ocean is doing (often flat).

A · Spots × time heatmap Dense and scannable, whole forecast in one view. Quality encoded twice — colour and bar height — with a greyscale toggle to prove it. B · Per-spot sparklines over the drivers Answers why. Every spot's line sits above the same swell / wind / tide charts, so you can watch spots disagree about identical inputs. C · Best now / today / this week The phone answer. One hero number, a ranked list, tap to drill in. Colour carries no information here at all.

Serve over HTTP (fetch is blocked on file://):
python3 -m http.server 8000 then open localhost:8000/demos/
